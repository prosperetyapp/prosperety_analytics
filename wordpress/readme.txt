== Description ==

Prosperty Analytics plugin allows to track the user actions on the webpage and sends them to the Propsperety server where the relevant action is taken depending on the marketing campaigns in progress.

== Frequently Asked Questions ==
Q1. How do I enable my website with prosperety Analytics
Ans. You need to first register you business on www.prosprety.com. After that you can enable the prosperety analytics plugin on your website and start enjoying the power of marketing campaigns powered by the propserety platform.

Q2. What are the steps to integrate the prosperety plugin
Ans. The following steps are required
        1) Install Prosperety-Analytics Plugin and activate it.
        2) Put shortcode in your website: from admin side
        3) Eg. [pAnalyticsInitialize]
        4) After adding shortcodes go to the website and hit the URL with the username parameter.
